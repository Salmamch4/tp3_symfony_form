# tp3_symfony_form

Ce projet consiste à transformer un formulaire HTML en formulaire Symfony.

Travail réalisé :
- Création d’un formulaire avec FormType
- Utilisation de Twig pour personnaliser l’affichage
- Intégration de Bootstrap 5.3
- Gestion du formulaire via un Controller Symfony
