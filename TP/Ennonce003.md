# TP 3 - Les formulaires en Symfony

## Objectif
Créer et personnaliser un formulaire en Symfony
en utilisant Twig et les Form Types.

## Travail demandé
- Transformer un formulaire HTML en formulaire Symfony
- Utiliser un Controller
- Utiliser un FormType sans entité
- Utiliser Twig pour l’affichage
- Intégrer Bootstrap

## Remarque
Ce repository contient uniquement le code du TP 3
conformément aux consignes de l’enseignant.
